# Project Purpose: Quick Notes
"Quick Notes" is a lightweight, distraction-aware note-taking browser extension. It allows users to keep notes while browsing via a floating panel (injected into any page) or a toolbar popup.

## Key Features
- Floating, draggable, and resizable panel (Shadow DOM).
- Auto-saves to `chrome.storage.local`.
- Markdown support with live preview.
- Multiple notes with pinning and color tags.
- Split view for side-by-side editing.
- Google Drive sync (opt-in).
- Search across all notes.
- History of clipboard pastes and URL captures.
