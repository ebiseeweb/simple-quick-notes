📝 Quick Notes · Options

Configure auto-paste, pinned floating panel, and backup / restore.

⚡ Auto-paste sites

When the popup opens on a matching URL, the clipboard is auto-appended to the active note (same content is only pasted once per clip).

    🪟 Floating panel · pinned sites

    On matching URLs, the floating Quick Notes panel auto-shows when the page finishes loading. You can also pin/unpin the current page directly from the panel's 📌 button.

      Tip: pattern syntax uses * wildcards. Examples: *://*.example.com/*, https://site.com/path/*. Toggle a panel manually on any page with Alt+Shift+N.

      🎯 Per-site default notes

      When the popup opens on a matching URL, this note is auto-selected. Set/unset a default directly from the editor via the 🎯 Default button.

        ◆ Popup shape

        Visual shape of the floating panel and standalone editor. Rescue option if a shape made the shape picker hard to reach.

        🔘 Default view

        Choose how the extension opens by default when you click the icon or press Alt+N.

        Tip: Popup Mode is best for sites that block floating panels. Float Focus Mode automatically clicks into the editor for you.

        ◑ Panel opacity

        Background transparency of the floating panel and standalone editor. Text and icons remain fully opaque so readability is preserved.

        100%

        🎨 Custom theme colors

        Override the default palette for the dark and light themes. Changes apply to the floating panel and popup. Only five essential colors are exposed — other shades (borders, hovers, etc.) derive automatically.

        💾 Backup & restore

        Export everything · notes, history, settings, theme · to a single JSON file. Import to restore on another machine or recover after a clear.

        ☁ Google Drive sync

        Sync notes, history, images, and settings to your Google Drive's hidden app-data folder. Invisible to you in your Drive UI; only this extension can access it. Use it to back up and restore across machines.
